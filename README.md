## Proyecto final unidad 1

Este proyecto consiste en el desarrollo de una página web del clima utilizando HTML semántico. 
La aplicación muestra información básica del clima como la ciudad, temperatura actual, 
sensación térmica, humedad, viento y precipitación. También incluye un pronóstico diario 
y un pronóstico por horas con iconos representativos del estado del tiempo.

El objetivo del proyecto es practicar la estructura básica de HTML, el uso de etiquetas 
semánticas y la organización del contenido para futuras funcionalidades dinámicas con JavaScript.

---

## Características

- Uso de HTML semántico (header, main, section, footer)
- Buscador de ciudades
- Selector de unidades (Celsius y Fahrenheit)
- Visualización de temperatura actual
- Detalles del clima:
  - Sensación térmica
  - Humedad
  - Viento
  - Precipitación
- Pronóstico diario con iconos
- Pronóstico por horas
- Elementos preparados para ser dinámicos usando `id`
- Uso de imágenes como iconos del clima

## Cómo ejecutar el proyecto

1. Descargar o clonar el repositorio.
2. Abrir la carpeta del proyecto.
3. Abrir el archivo `index.html` en cualquier navegador web.

O si usas VS Code:
1. Abrir la carpeta en Visual Studio Code.
2. Instalar la extensión **Live Server** (opcional).
3. Ejecutar el archivo `index.html`.
